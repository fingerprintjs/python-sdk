# SearchEventsBotInfo
Filter events by their Bot Info result, specifically:
  - `all` - events where any kind of bot was detected.
  - `none` - events where no bot was detected.


## Enum

* `ALL` (value: `'all'`)
* `NONE` (value: `'none'`)
