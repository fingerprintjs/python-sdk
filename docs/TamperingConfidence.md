# TamperingConfidence
Confidence level of the tampering detection.
If a proxy is not detected, confidence is "high".
If it's detected, can be "low", "medium", or "high".


## Enum

* `LOW` (value: `'low'`)
* `MEDIUM` (value: `'medium'`)
* `HIGH` (value: `'high'`)
